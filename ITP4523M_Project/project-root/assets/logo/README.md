# Logo Assets Directory

This directory contains all logo variations for Premium Living Furniture.

## Logo Files

### Main Logos
- `logo-dark.png` - Dark version for light backgrounds (200x50px)
- `logo-light.png` - Light version for dark backgrounds (200x50px)
- `logo-dark@2x.png` - Retina dark version (400x100px)
- `logo-light@2x.png` - Retina light version (400x100px)

### Logo Elements
- `logo-icon.svg` - Icon only (couch symbol)
- `logo-text.svg` - Text only ("Premium Living")
- `logo-mark.svg` - Brand mark/emblem

### Alternative Versions
- `logo-square.png` - Square version for social media
- `logo-vertical.png` - Vertical stacked version
- `logo-white.png` - All white version
- `logo-black.png` - All black version

## Logo Specifications

### Full Logo
- **Dimensions**: 200x50px (standard), 400x100px (retina)
- **Format**: PNG with transparency
- **Color**: Primary (#2D3748) and Secondary (#8B5A2B)

### Icon Only
- **Dimensions**: 80x80px (standard), 160x160px (retina)
- **Format**: SVG (preferred), PNG fallback
- **Symbol**: Stylized couch/furniture icon

### Text Only
- **Dimensions**: 150x30px (standard), 300x60px (retina)
- **Font**: Inter Bold
- **Kerning**: Optimized for legibility

## Usage Guidelines

### Light Backgrounds
Use `logo-dark.png` or `logo-dark@2x.png` on white/light backgrounds.

### Dark Backgrounds  
Use `logo-light.png` or `logo-light@2x.png` on dark/colored backgrounds.

### Favicon
Use `logo-icon.svg` converted to ICO format for browser tab.

### Social Media
Use `logo-square.png` (500x500px) for profile pictures.

## Color Variations

- **Full Color**: Primary and secondary brand colors
- **Monochrome Dark**: #2D3748 (dark gray)
- **Monochrome Light**: #FFFFFF (white)
- **Monochrome Black**: #000000 (black)

## File Naming Convention

`logo-[variant]-[color].[extension]`

Examples:
- `logo-full-dark.png`
- `logo-icon-light.svg`
- `logo-text-primary.png`
