# Product Images Directory

This directory contains all product images organized by category.

## Directory Structure

products/
├── sofas/ # All sofa and sectional images
├── tables/ # Coffee, dining, side tables
├── chairs/ # Armchairs, dining chairs, office chairs
├── lighting/ # Lamps, pendants, chandeliers
└── bedroom/ # Beds, nightstands, dressers


## Image Requirements

### Main Product Images
- **Resolution**: 1200x1200 pixels
- **Format**: JPG with 80% quality
- **Background**: White or transparent
- **File Size**: Under 500KB

### Gallery Images
- **Resolution**: 600x600 pixels
- **Format**: JPG with 80% quality
- **Angles**: Front, side, back, detail shots
- **File Size**: Under 200KB

### Thumbnails
- **Resolution**: 150x150 pixels
- **Format**: JPG with 70% quality
- **File Size**: Under 50KB

## Naming Convention

Format: `[product-name]-[variant]-[angle].[extension]`

Examples:
- `harmony-sofa-charcoal-main.jpg`
- `harmony-sofa-charcoal-side.jpg`
- `harmony-sofa-charcoal-back.jpg`
- `harmony-sofa-charcoal-detail.jpg`

## Required Images Per Product

1. **Main Image** (required) - `[product-name]-main.jpg`
2. **Alternate Angle** (recommended) - `[product-name]-side.jpg`
3. **Detail Shot** (recommended) - `[product-name]-detail.jpg`
4. **In-Situ** (optional) - `[product-name]-lifestyle.jpg`

## Color Variants

For products with multiple colors:
`[product-name]-[color]-main.jpg`

Example:
- `harmony-sofa-charcoal-main.jpg`
- `harmony-sofa-beige-main.jpg`
- `harmony-sofa-navy-main.jpg`
